# 几个重要建站名词的个人理解

# 基本关键词解释

# 服务器：

其实服务器也是一台特殊的电脑，我们上网的过程就是从服务器取得网页页面的过程。打开网页的过程可理解成是取得服务器上面的一个文件，大多数时候是html文件，或者动态生成的html文件。

# 域名：

但是服务器的地址是ip形式的，我们平时可是通过www网址的形式访问的。这里说到的www.XXX.com就是域名，可以理解为服务器地址的一个别名，域名不止这一种形式，还有.cn 、 .org 、 .net 等后缀。

# DNS服务器

我们输入网址之后，不会立刻带到目标服务器（除非是输入ip地址），而是要先经过DNS服务器。DNS服务器上面保存了世界上所有的网站的域名和ip地址的对应关系。

所以是输入网址 -> DNS服务器将域名解析成ip地址 -> 连接上服务器 -> 服务器返回结果 ，通过这样的过程，我们就可以打开相应的网页了。

# 建站需要什么

主要来讲，只需要两个东西：域名和空间。

域名的作用上面提到了，就是提供一个简单易记的访问方式。空间就是用来储存网页的地方，当然，也可以用来运行服务器程序，动态生成页面。

买了域名和空间之后，分别设置。让域名链接到空间的ip地址，同时在空间处设置，允许你购买的域名能够转向你的空间。就是双向允许的设置。

# 什么是备案

备案是大陆才有的，备案是针对域名的说法，和空间无关。

说说不备案有什么坏处吧：不备案的话，大陆地区的服务器是不允许（不是不能）绑定未备案的域名的，所以这些域名只能选择海外或者香港的服务器空间。

以上是近期学习到的相应的知识，希望对刚刚建站的朋友有所帮助。
